## 🚀 Quick Start Commands

### Start V2 Application:
```powershell
# Start all services (database, API, AI engine, frontend)
docker-compose up

# OR start individually for development:

# Terminal 1: AI Engine
cd ai-engine
..\.venv\Scripts\activate
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload

# Terminal 2: Frontend
cd frontend
npm run dev

# Terminal 3: API Gateway (optional)
cd api-gateway
npm run dev
```

### Access V2 Application:
- **Frontend:** http://localhost:5173
- **AI Engine:** http://localhost:8000
- **API Gateway:** http://localhost:3001

## 🏗️ V2 Architecture Pattern

**Industry Standard: Microservices with AI-First Design**

1. **Frontend** (React SPA) → User uploads PDFs
2. **API Gateway** (TypeScript) → Handles HTTP, manages SQL Server
3. **AI Engine** (Python) → LangGraph agents process documents
   - Manager Agent (orchestrator)
   - Extraction Agent
   - Quality Review Agent
   - Reconciliation Agent
   - 6 Helper Agents (OCR, Table Parser, etc.)

**Communication:**
- Frontend ↔ API Gateway (REST)
- API Gateway ↔ AI Engine (REST + WebSocket for real-time chat)
- AI Engine → PostgreSQL (LangGraph checkpoints)
- API Gateway → SQL Server (business data)
- All → Azure Blob Storage (PDF documents)
